# Address Book Management System

## Description

A menu-driven Address Book Management System developed using C programming.

This project helps to manage contact information efficiently by using structures, file handling, and modular programming concepts.

## Features

- Create new contacts
- Search contacts
- Edit contact details
- Delete contacts
- Display all contacts
- Store contacts permanently using file handling
- Data validation

## Technologies Used

- C Programming
- Structures
- File Handling
- Pointers
- Modular Programming

## Project Structure

```
AddressBook
│
├── main.c          # Main program flow
├── contact.c       # Contact management functions
├── contact.h       # Function declarations
├── file.c          # File handling operations
├── file.h
│
└── contacts.txt    # Contact database
```

## Concepts Learned

- Structure implementation
- Function-based programming
- File operations
- Data management
- Searching and updating records

## How to Run

Compile:

```bash
gcc *.c -o addressbook
```

Run:

```bash
./addressbook
```
